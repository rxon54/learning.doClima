## Core Concepts

- **Mann-Kendall test** — non-parametric trend test for non-normally distributed data (most climate data)
- **Sen's slope** — robust trend magnitude estimator, less sensitive to outliers than linear regression
- **Break point detection** — station moves, instrument changes, or land use shifts create discontinuities that must be detected and corrected (homogenisation)
- **Baseline period** — "normal" is currently 1991-2020; changes every decade
- **Autocorrelation** — climate data has memory (wet years follow wet years); must account for this in significance tests
