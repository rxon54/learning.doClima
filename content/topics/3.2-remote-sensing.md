## Core Concepts

- **Key missions:** [[landsat]] (30m, 1972—), [[sentinel2]] (10m, 2015—), [[modis]] (250m, daily)
- **[[ndvi]]** — (NIR - Red) / (NIR + Red). Healthy vegetation vs bare soil.
- **NBR** — Normalized Burn Ratio. Identifies burned areas.
- Annual composites — MapBiomas uses best-pixel-per-year to avoid cloud cover
- Classification: MapBiomas uses machine learning trained on manually classified samples
