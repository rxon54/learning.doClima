# NDVI — Normalized Difference Vegetation Index

(NIR - Red) / (NIR + Red). Values range -1 to +1. Healthy dense vegetation: 0.6–0.9. Sparse vegetation: 0.2–0.4. Bare soil: ~0.1. Water: negative.

**Why it works:** healthy chlorophyll strongly absorbs red light (for photosynthesis) and strongly reflects near-infrared (cell structure). Stressed or absent vegetation breaks this pattern.
