# SIRGAS 2000 — Brazil's Official Geodetic Reference System

Brazil's official geodetic datum since 2015 (fully replaced Córrego Alegre). EPSG:4674. Aligned with ITRS (International Terrestrial Reference System) — essentially matches WGS84 at the centimeter level for most practical purposes.

**Why it matters:** All official Brazilian spatial data (IBGE boundaries, ANA hydrological data, AdaptaBrasil shapefiles) uses SIRGAS 2000. When combining with international datasets (WGS84/EPSG:4326), the difference is negligible at municipal scale but should be accounted for in precision work.
