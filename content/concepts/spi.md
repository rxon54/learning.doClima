# SPI — Standardized Precipitation Index

Probabilistic drought index developed by McKee et al. (1993). Transforms precipitation data into a standard normal distribution so drought severity can be compared across regions with very different rainfall regimes.

**Scale:** +2 = extremely wet, 0 = normal, -1 = moderately dry, -1.5 = severely dry, -2 = extremely dry (drought emergency threshold).

**Time scales:** SPI-1 (1-month) = meteorological drought; SPI-3 (3-month) = agricultural drought onset; SPI-12 (12-month) = hydrological drought / reservoir impacts.

**Key advantage:** 50mm below normal in the Amazon and 50mm below normal in the Sertão are very different things — SPI normalizes for local climatology so they're comparable.
